# Travel Planner

Travel Planner es una aplicación web para organizar viajes y preparar la información necesaria antes de salir.

El objetivo del proyecto es centralizar en un único sitio los datos principales de cada viaje, evitando depender de notas sueltas, capturas, correos o mensajes repartidos en distintas aplicaciones.

## Descripción

La aplicación permite gestionar viajes desde una interfaz web sencilla.

Cada viaje puede contener información básica como destino, fechas y descripción. El frontend se comunica con un backend mediante una API REST, y los datos se almacenan en una base de datos PostgreSQL.

## Tecnologías principales

* Java 17
* Spring Boot
* PostgreSQL
* React
* TypeScript
* Vite
